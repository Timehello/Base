    # 所有属性

    属性 类型 默认值 说明
    items Array, Object [] 选项数据。组件通过 trackBy 和 labelBy 来解析
    trackBy String 'key' 定义选项数据索引字段
    labelBy String 'label' 定义选项数据的文本字段
    isMulti Boolean false 是否为多选
    multiType String 'string' 定义多选情况下，组件如何处理 v-model 绑定的数据类型。可选 'array'，为数组
    resetText String -- 定义重置选项的文本
    resetValue null null 定义重置选项的索引
    canReset Boolean true 定义是否显示重置按钮；如果设置了重置选项，组件将不提供重置按钮功能
    autoHeight Boolean false 打开后，内容一行显示不完会自动撑开组件高度
    size Number 1 定义组件尺寸，默认1，可选2，小尺寸
    inline Boolean false 定义组件是否为行内
    disabled Boolean false 定义组件是否禁用
    placeholder String -- 定义组件的 placeholder
    zIndex Number -- 定义下拉弹层的 zIndex
    disableSearch Boolean false 是否显示搜索框
