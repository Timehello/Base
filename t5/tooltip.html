    # 文字提示 Tooltip

    TIP

    文字提示气泡框，在鼠标悬停时显示，代替了系统的title提示。

    提示框出现的位置，通过 placement 配置，可选值为top、top-start、top-end、bottom、bottom-start、bottom-end、left、left-start、left-end、right、right-start、right-end。

    # 禁用

    通过设置属性disabled可以禁用文字提示。

    Can disable text prompts


    <template>
    <Tooltip placement="top" content="Can disable text prompts" :disabled="disabled">
        <Button type="2">禁用文字提示</Button>
    </Tooltip>
</template>

<script>
  import { Tooltip, Button } from 'talebase-ui'
  export default {
    components: {
      Tooltip,
      Button
    },
    data () {
        return {
            disabled: true
        }
    }
  }
</script>

    # 自定义内容

    通过自定义 slot 显示多行文本或更复杂的样式。

    广东倍智测聘网络科技股份有限公司成立于2011年1月18日，总部设在广州，在全国有11个分支机构，200多名员工

    自定义样式



    <template>
    <div>
        <Tooltip placement="top">
            <Button>多行文本</Button>
            <div slot="content">
                <p style="white-space: normal;">广东倍智测聘网络科技股份有限公司成立于2011年1月18日，总部设在广州，在全国有11个分支机构，200多名员工</p>
                <p><i style="font-size:18px;color:red">自定义样式</i></p>
            </div>
        </Tooltip>
    </div>
</template>
<script>
  import { Tooltip, Button } from 'talebase-ui'
  export default {
    components: {
      Tooltip,
      Button
    }
  }
</script>

    # API

    属性 类型 默认值 说明
    placement String bottom 提示框出现的位置，可选值为top、top-start、top-end、bottom、bottom-start、bottom-end、left、left-start、left-end、right、right-start、right-end
    content String,Number 空 显示的内容
    disabled Boolean false 是否禁用提示框
    theme String dark 主题，可选值为 dark 或 light、yellow
    max-width String,Number - 最大宽度，超出最大值后，文本将自动换行，并两端对齐
    delay Number 0 延迟显示，单位毫秒
    offset Number 0 出现位置的偏移量
    always Boolean false 是否总是可见
    transfer Boolean false 是否将弹层放置于 body 内，在 Tabs、带有 fixed 的 Table 列内使用时，建议添加此属性，它将不受父级样式影响，从而达到更好的效果
    options Object 自定义 popper.js 的配置项，具体配置见 popper.js 文档
    事件名 说明 返回值 使用
    on-popper-show 在提示框显示时触发 无 @on-popper-show="xxx"
    on-popper-hide 在提示框消失时触发 无 @on-popper-hide="xxx"
