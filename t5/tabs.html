# 选项卡 tabs

选项卡是为同范畴内容提供不同信息展示切换的组件，具有容纳、导航作用。

# 基本用法






<tabs v-model="activeName" :type="3">
  <tab-item label="苹果（导航）" to="http://hcysun.me/vue-design/art/1start-learn.html">苹果</tab-item>
  <tab-item label="橄榄" >橄榄</tab-item>
  <tab-item label="梅子" >梅子</tab-item>
  <tab-item >
    <span slot="label"><i>icon</i>香蕉</span>
    <div>大把大把的焦 水果中的挚爱</div>
  </tab-item>
</tabs>

TIP

tabs采用v-model形式 绑定的值value是当前tab的name或者下标

# API

属性

# tab
属性 类型 默认值 说明
type Number 1 1 2 3对应以上3种样式
# tab-item
属性 类型 默认值 说明
label Number 1 1 2 3对应以上3种样式
name Sting Number 可选 绑定activeName 不传默认使用下标
to Sting 当路由跳转时的路径
