    TIP

    侧滑框可在页面中叠加使用
    键盘Esc键可关闭侧滑框

    # API

    插槽

    命名 说明
    default 侧滑框中间内容部分
    title 自定义侧滑框抬头
    footer 自定义侧滑框脚部

    属性

    属性 类型 默认值 说明
    visible Boolean false 是否显示侧滑框
    disabled Boolean false 是否禁用确认按钮
    size String 'normal' 侧滑框大小 可选 'bg'中等，'lg'大
    hideFooter Boolean false 是否需要脚部按钮
    cancelable Boolean true 是否允许关闭侧滑框
    fixScrollY boolean false

    事件

    事件名 说明 返回值
    open 打开侧滑框时触发
    close 关闭侧滑框时触发
    cancel 点击侧滑框外部或上下关闭按钮触发
    confirm 点击侧滑框底部确认按钮触发
