    # 步骤条 Step

    拆分某项流程的步骤，引导用户按流程完成任务。

    # 基本用法


    1 步骤一 1 步骤二 1 步骤三 1 步骤四 1 步骤五

    # 文案

    默认没有标题与步骤文本，需要根据不同场景设置。

    步骤条 1 步骤一 1 步骤二 1 步骤三 1 步骤四
    < 右边可自定义


    <step label="步骤条" :active.sync = "active">
  <step-item>自定义的步骤一</step-item>
  <step-item>自定义的步骤二</step-item>
  <step-item>自定义的步骤三</step-item>
  <step-item>自定义的步骤四</step-item>
</step>

data () {
  return {
    active: 0
  }
}

    # API

    属性

    Step

    属性 类型 默认值 说明
    float Boolean false 组件是否左浮动，默认呈块状布局。仅在块状布局下可以定义右边slot。
    label String '' 步骤条的标题
    type Number 1 步骤条样式，1为填充箭头，2为连接线条
    active Number 0 当前步骤，从 0 开始计数，代表着第一步高亮；须搭配sync语法糖，并且赋值给一个变量

    StepItem

    属性 类型 默认值 说明
    URL String '' （根据路由规则设置）如果有值，则当前页面是该URL时该步骤会高亮
    visible Boolean true 控制具体步骤是否展示
