    # button

    # 尺寸

    属性是 size, 2为小尺寸，其余为普通尺寸

    <t-button>size 1</t-button>
<t-button :size="2">size 2</t-button>

    # 类型

    属性是type，有3个可选值，1，2，3；默认1

    <t-button>色块按钮</t-button>
<t-button type="2">线框按钮1</t-button>
<t-button type="3">线框按钮2</t-button>

    # 禁用状态

    属性是disabled，布尔，默认false

    <t-button disabled>下单</t-button>

    # 块状按钮

    属性是 block，默认 false

    <t-button block>确定</t-button>

    # 标签

    属性是 tag，默认是 button，可以换成其他标签

    确定 
    <t-button tag="span">确定</t-button> // 渲染出来就是 span 标签

    # 点击事件

    会直接绑定在按钮元素上

    # 所有属性

    属性 类型 默认值 说明
    tag String 'button' 按钮渲染的标签
    disabled Boolean false 是否禁用
    size String,Number 1 2为小尺寸，其余为普通尺寸
    type String,Number 1 1：色块按钮，2：线框按钮1，3：线框按钮2
    block Boolean false 是否为块状

    # 事件

    支持渲染标签所有事件

    TIP

    如果按钮禁用，click事件也会不响应
