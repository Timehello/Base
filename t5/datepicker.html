    # 所有属性

    属性 类型 默认值 说明
    placeholder String -- placeholder
    disabled Boolean false 禁用
    size Number 1 表单尺寸，默认1，可选2（小尺寸）
    inline Boolean false 是否行内
    canReset Boolean true 能否重置日期
    disabledDate Function -- 回调函数，接收一个日期，返回布尔值表示这个日期能否选择
    type String 'date' 切换选单个日期和范围，范围是 ‘range’
    showTime Boolean false 能否选择时间
    zIndex Number -- 日期弹层的zIndex
    loose Boolean false 是否可以单选一个时间。仅 type === 'range' 时有效
